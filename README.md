# Advent of Code in Java 

## Setup

1. Make sure you clone this repo, and install java 21 :)
2. Add input data from the website to relevant date in the `resources` folder 
3. To create a new file for a new day, just copy previous java file, and call it day2.java, day3.java and so on...

## How to run 

Click run above the `main` method in intellij or eclipse
